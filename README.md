# ⚙️ KIPU Core - The Engine of the Empire

> El corazón lógico de KIPU. Diseñado para la alta disponibilidad en **Oracle Cloud Infrastructure (OCI)** y la gestión segura de activos digitales.

## 🏛️ Filosofía Core
Si la App es el nudo visible, **Kipu Core** es la cuerda que sostiene todo el sistema. Nuestra infraestructura se rige por:
* **Integridad de Datos:** Procesamiento de transacciones con validación rigurosa de tokens.
* **Escalabilidad en la Nube:** Preparado para el despliegue en instancias OCI con optimización de recursos.
* **Seguridad de Grado Bancario:** Implementación de flujos de Refresh Tokens y validación dinámica de sesiones.

## 🔧 Especificaciones Técnicas
* **Entorno:** Diseñado para Oracle Linux / Docker.
* **API:** RESTful Architecture.
* **Seguridad:** JWT (JSON Web Tokens) con rotación automática.
* **Persistencia:** Optimizado para bases de datos relacionales escalables.

## ☁️ Deployment (Oracle Cloud)
Este repositorio contiene los manifiestos necesarios para el despliegue en producción, asegurando que el "core" de la operación nunca se detenga.
